# Martel Inmobiliaria

Landing page de Martel Inmobiliaria — sitio web estático (HTML, CSS y JavaScript en un solo archivo `index.html`).

## Ver el sitio en local

Abre `index.html` en cualquier navegador (doble clic).

## Publicar en GitHub Pages

1. Sube este repositorio a GitHub.
2. En el repo: **Settings → Pages**.
3. En **Source**, elige **Deploy from a branch**.
4. Selecciona la rama `main` y la carpeta `/ (root)`. Guarda.
5. En 1–2 minutos el sitio estará en `https://<usuario>.github.io/<repo>/`.
